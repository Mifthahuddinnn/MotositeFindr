## Getting Started

To get started running the project locally, please follow the steps below.

First, clone the repository.

```bash
git clone -b back-end https://github.com/fakihfavian24/MotositeFindr.git
```


```bash
cd MotositeFindr
npm install
```

Finally, run the development.

```bash
npm run index.js
```
Good luck.
